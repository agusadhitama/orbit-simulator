# ORBIT SIMULATOR

Sebuah **simulator orbit satelit sederhana di web** yang memungkinkan kamu mencoba menambah / menghapus satelit, mengatur kecepatan awal, dan melihat bagaimana satelit mengelilingi planet.

⚙️ Simulator ini dibuat dan ditampilkan oleh *Agus Satria Adhitama* dengan antarmuka minimalis yang fokus pada mekanika orbit secara visual.

## 📌 Deskripsi

**Orbit Simulator** adalah aplikasi web berbasis HTML/CSS/JavaScript yang mensimulasikan perilaku orbit satelit di sekitar sebuah pusat gravitasi (seperti Bumi).

Di antaranya kamu dapat:

- **Menambahkan (ADD) satelit** ke simulasi
- **Menghapus (REMOVE)** satelit yang sudah ditambahkan
- **Mengatur kecepatan (SPEED)** awal satelit  
- Melihat jumlah satelit yang sedang aktif ditampilkan

Tampilan dan kontrol dibuat sederhana supaya mudah dipakai sambil belajar dasar‑dasar **mekanika orbit**.

## 🛰️ Cara Menggunakan Simulator

1. **Buka halaman web** di browser.
2. Klik tombol **ADD SAT** untuk menambah satelit baru.
3. Atur nilai **SPEED** untuk satelit yang ditambahkan.
4. Jika ingin menghapus satelit, klik **REMOVE SAT**.
5. Amati lintasan orbit dan perilaku satelit di area simulasi.
